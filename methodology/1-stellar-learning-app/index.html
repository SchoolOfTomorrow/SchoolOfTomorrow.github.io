The stellar-learning.app

Stellar-learning.app is based on empirically proven teaching and learning strategies from the research area of learning and memory in cognitive psychology: distributed learning, recall training, corrective feedback and multi-sensory coding. The application was developed by Distance University Switzerland in cooperation with the software company JLS DIGITAL as part of the project: School of Tomorrow. It is meant to be used as additional learning and teaching tool when introducing a foreign language.

Design of the app

Stellar-learning.app uses a space-related theme to motivate a child-friendly learning environment. It enables the use in several settings, i.e., in classroom, at school, at home or anywhere else, by choosing whether a microphone or speakers are turned on or off for a learning session.

Implemented learning principles

Distributed learning

Recall training

Corrective feedback

Multi-sensory coding

How to get the app?

stellar-learning.app can be used in a web-based version or downloaded via the app-store. If you like to test the language learning app, contact us!
